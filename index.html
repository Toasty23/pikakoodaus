HTML

HTML on avoimen standardin mukainen merkintäkieli, jota käytetään verkkosivujen rakentamiseen. HTML:llä voidaan luoda sivujen rakenteellinen kuvaus, kuten otsikot, tekstit, linkit ja kuvat.

CSS

CSS (Cascading Style Sheets) on tyylisivukieli, jota käytetään verkkosivujen ulkoasun määrittämiseen. CSS:llä voidaan muotoilla HTML-elementtejä, kuten taustavärit, fontit, marginaalit ja animaatiot.

JavaScript

JavaScript on skriptauskieli, jota käytetään pääasiassa verkkosivujen dynaamisuuden toteuttamiseen. JavaScriptillä voidaan luoda interaktiivisia toiminnallisuuksia, kuten tapahtumankäsittelyä ja AJAX-pyyntöjä.

Python

Python on korkean tason ohjelmointikieli, joka on suosittu monipuolisuutensa ja selkeän syntaksinsa ansiosta. Pythonilla voidaan toteuttaa erilaisia sovelluksia, kuten web-sovelluksia, tietojenkäsittelyä ja tekoälyä.

C++

C++ on tehokas ohjelmointikieli, joka perustuu C-kieleen. C++:lla voidaan toteuttaa nopeaa ja suorituskykyistä ohjelmistokehitystä, kuten järjestelmäohjelmointia, peliohjelmointia ja sulautettuja järjestelmiä.
